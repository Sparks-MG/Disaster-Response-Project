Information on the Model behind the Classification

The classifcation model was build on a multiclassifcation extended randomforest algorithm as implemented in Pythons Scikit-learn library. The hyperparameters were carefully tuned with Scikit-learns Gridsearch. The number of trees was limited to 100, as the metrics indicate only minor improval for a larger number of trees while generating a larger binary. The corresponding Github Respository is linked above at the contact tab.
